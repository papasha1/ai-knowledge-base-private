# 快速录入模板

> 复制以下模板，填入对应文件即可。带 * 的为必填，其他可选。

---

## 最简版（3 个字段）

```
## 标题 *

**记忆点：** 一句话记住核心，像口诀一样 *

**掌握程度：** 🟡（🔴只是知道 / 🟡理解实操过 / 🟢能独立解决）
```

## 标准版（5 个字段）

```
## 标题 *

**记忆点：** 一句话口诀 *

**掌握程度：** 🟡

**场景：** 什么情况下遇到的 *

**原理：** 通俗解释 *
```

## 完整版（8 个字段）

```
## 标题 *

**记忆点：** 一句话口诀，回忆时最先想到的那句 *

**掌握程度：** 🟢 / 🟡 / 🔴 *

**场景：** 什么情况下遇到的 *

**原理：** 通俗解释 *

**代码：** `项目路径/文件:行号`

**关联：** [相关条目](文件.md#标题)

**来源：** 项目名，日期 *
```
